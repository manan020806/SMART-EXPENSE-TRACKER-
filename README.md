# Smart Expense Tracker 💰

## 📌 Description
A simple Java console application that helps users track daily expenses.

## 🚀 Features
- Add expenses
- View all expenses
- Calculate total spending
- Filter expenses by category

## 🛠️ Technologies Used
- Java
- OOP concepts
- ArrayList

## ▶️ How to Run
1. Compile the code:
   javac ExpenseTracker.java

2. Run:
   java ExpenseTracker

## 📷 Sample Output
===== Expense Tracker =====
1. Add Expense
2. View Expenses
3. Total Spending
4. Filter by Category
5. Exit
